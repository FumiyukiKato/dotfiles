# dotfiles
configuration files

### Install

```
$ brew update
$ brew install zsh

$ sudo vi /etc/shells # add "/usr/local/bin/zsh"
$ chsh -s /usr/local/bin/zsh
$ zsh

$ cd dotfiles/
$ bin/zsh_install
```
