# dotfiles
configuration files

### Install

```
$ brew update
$ brew install zsh

$ sudo echo "/usr/local/bin/zsh" >> /etc/shells  # (vi /etc/shells)
$ chsh -s /usr/local/bin/zsh
$ zsh

$ cd dotfiles/
$ bin/zsh_install
```
